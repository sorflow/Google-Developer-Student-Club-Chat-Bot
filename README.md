## ğŸš€ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- Python 3.x
- npm or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/AAMU-GDG-AI-Chatbot.git
   cd AAMU-GDG-AI-Chatbot
   ```

2. Install frontend dependencies:
   ```bash
   cd frontend
   npm install
   # or
   pnpm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   pnpm dev
   ```
